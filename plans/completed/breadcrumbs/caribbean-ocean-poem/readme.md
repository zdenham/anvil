# Caribbean Ocean Poem

## Objective
Write a short, evocative poem about the Caribbean ocean. The poem should capture the beauty, warmth, and spirit of the Caribbean sea.

## Acceptance Criteria
- A complete poem (8-16 lines)
- Themed around the Caribbean ocean — its colors, warmth, waves, culture
- Should feel finished and polished in a single iteration

## Context
This is a simple creative writing task. No codebase changes needed. Just produce the poem and mark complete.
